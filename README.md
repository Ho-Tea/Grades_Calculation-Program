### Scenario #2 (클래스 정의)  

클래스(멤버변수) :
- 학생클래스(학번, 이름, 전공)
- 과목클래스(과목, ...+)
- 성적클래스(성적, ...+)
- 전공클래스(...+)


-> 학생클래스는 변경가능성이 적으므로 인터페이스 추가 X  
-> 과목클래스는 변경가능성이 적으므로 인터페이스 추가 X  
-> 성적클래스는 추후에 성적산출방법이 바뀔 수 있으니 인터페이스 추가  
-> 전공클래스는 추후에 전공 별 필수과목이 바뀔 수 있으니 인터페이스 추가
